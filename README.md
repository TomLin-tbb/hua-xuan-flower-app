🌸 花选平台——鲜花选购与养花记录小程序

📌 项目简介
花选是一款主打「简单选花，轻松记录」的鲜花类小程序，分为购花商城与养花台账两大模块，兼顾用户购花消费与居家养花记录的双重需求，界面简洁易用，操作逻辑清晰。

🛠️ 技术栈
开发工具：HBuilderX
前端框架：uni-app（跨平台，可打包成小程序/App）
后端服务：uniCloud（阿里云版，云数据库存储数据）
数据存储：JSON静态数据 + 云数据库
组件库：uni-ui 组件库

✨ 核心功能与流程
1.首页（index）：展示花卉商品列表，标注价格、生长习性，支持关键词检索与加购操作。
2.登记页（form）：用户可录入花品信息、入手日期、养护方案并上传图片，形成可编辑、删除的专属养花台账。
3.个人中心（mine）：整合购物车、订单管理、意见反馈、账号设置功能，展示不同状态订单，支持售后与评价操作。
4.数据存储：通过 uniCloud 云数据库存储花卉信息与用户养护数据，实现数据持久化。
